Props

Props stands for properties.

Props are arguments passed into React components.

Components can be passed as props.

Props are like function arguments, and you send them into the component as HTML attributes.

Components in Files

It is recommended to split your components into separate files.

To do that, create a new file with a .js file extension and put the code inside it.

The filename must start with an uppercase character.

This example creates a Component named Car.js.

function Car() {
return <h2>Hi, I am a Car!</h2>;
}

export default Car;

You can then import it into our application.

import Car from './Car.js';
